=== OpenPDC base ===
Contributors: Yard Internet
Requires at least: 4.8
Tested up to: 5.1
Requires PHP: 7.0
Stable tag: 2.1.7

Plugin to add the OpenwebConcept OpenPDC to your project.

It enables the input of PDC (Producten Diensten Catalogus), and adds rest api endpoints.
